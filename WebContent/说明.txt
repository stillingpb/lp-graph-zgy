1.index.html页面就是关系图的页面，是基于HTML5技术实现的，所以浏览器需要支持HTML5的。这个页面是直接调用后台的maps文件夹下的JSON格式的数据文件的，所以可能很方便地把这些JSON文件换成后台的数据输出接口，具体的数据输出接口，源代码中也给出了样例。

2.关系图的JSON数据格式具体可以参考maps文件夹下的JSON文件，主要是包含两方面的内容，一方面是节点数据nodes，另一方面是节点与节点的连线数据edges。具体的格式如下：
{
    "nodes": {
        "节点KEY1": {key:value,key:value},
        "节点KEY2": {},
        "节点KEY3": {}
    },

    "edges": {
        "节点KEY1": {
            "节点KEY2": {},"节点KEY3": {}
        },
        "节点KEY2": {
            "节点KEY3": {}
        }
    }
}

3.这个的实现采用的是开源的HTML5框架，有什么具体的问题，可以联系我。
